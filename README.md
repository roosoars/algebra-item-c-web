# ALGEBRA LINEAR - UFU

Versao web para calcular as transformacoes do item C de Algebra Linear.

## Repositorio

- https://github.com/roosoars/algebra-item-c-web

## GitHub Pages

- URL esperada: https://roosoars.github.io/algebra-item-c-web/
- Deploy automatico via GitHub Actions em cada push na branch `main`.

## Requisitos

- Node.js 20+
- npm 10+

## Desenvolvimento

```bash
npm install
npm run dev
```

## Testes

```bash
npm run test
```

## Build de producao

```bash
npm run build
npm run preview
```
